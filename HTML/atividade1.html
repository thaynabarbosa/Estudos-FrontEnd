Fatos sobre HTML

A seguir alguma características de HTML

  • Não é um alinguagem de programação
  • Espaços em texto escrito não necessariamente será visualizado como espaço na página
  • Quebras de linha apenas usando tags específicas, como "p" e "br"
  • O nome da tag "a" vem de "anchor", ou "âncora"
  • O atributo "target=_black" da tag "a" abre o link em nova aba ou janela
